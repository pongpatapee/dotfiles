# My Dotfiles

dotfiles will now be managed via GNU/stow

Stow [video](https://www.youtube.com/watch?v=y6XCebnB9gs)

## Requirements

### Git

```
sudo apt install git
```

### Stow

```
sudo apt install stow
```

## Installation

```
cd $HOME

git clone https://github.com/pongpatapee/dotfiles.git

cd dotfiles

stow .
```
